# Skin-Cancer-Detection-System-Using-CNN-Deep-Learning-
This project is a deep learning-based application that detects whether a skin lesion is Benign or Malignant using image classification.

I built this project to understand how Convolutional Neural Networks (CNNs) can be used in real-world healthcare problems.

## How it works
The user uploads a skin image
The image is processed and resized
A trained CNN model predicts the result
Output: Benign or Malignant

## Features
Image upload using Streamlit
Real-time prediction
Simple and user-friendly interface

## Model
CNN (Convolutional Neural Network)
Input size: 224 × 224 images
Output: Binary classification (Benign / Malignant)

## Tech Stack
Python
TensorFlow / Keras
NumPy
Matplotlib
Streamlit
